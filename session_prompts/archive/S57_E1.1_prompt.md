## FELADATOK

### LEPES 1: ImapBackend + EmailSourceAdapter core

```
Cel:    IMAP folder polling + MIME parse → IntakePackage (valos imaplib + email.parser).
Forras: 01_PLAN/session_S55_*.md Day 2 (E1.1-A)
Fajlok:
  - src/aiflow/sources/email_adapter.py (uj)
      * EmailSourceAdapter(SourceAdapter)
        - source_type = IntakeSourceType.EMAIL
        - metadata: transport="pull", requires_ack=True, supports_batching=False
      * ImapBackend (sync imaplib.IMAP4_SSL + asyncio.to_thread wrapper)
        - connect(host, port, user, password, mailbox="INBOX")
        - fetch_unseen() -> list[tuple[uid, raw_bytes]]
        - mark_seen(uid: int) / mark_flagged(uid: int, reason: str)
      * MIME parser (email.message_from_bytes):
        - subject / from / body → IntakeDescription(role="email_body")
        - attachments → IntakeFile (file_path written under storage_root)
        - charset fallback: utf-8 → latin-1 → replace

API:
  adapter = EmailSourceAdapter(backend=ImapBackend(...), storage_root=Path(...), tenant_id="...")
  pkg = await adapter.fetch_next()       # None if no unseen
  await adapter.acknowledge(pkg.package_id)   # flags \Seen
  await adapter.reject(pkg.package_id, reason="oversize")  # flags \Flagged + stores reason
```
