## FELADATOK

### LEPES 1: `FolderSourceAdapter` implementacio

```
Cel:    watchdog.Observer → debounce → stable-mtime guard → enqueue(IntakePackage)
Fajl:   src/aiflow/sources/folder_adapter.py (uj)
Forras: 01_PLAN/session_S55_*.md Day 6 (E2.1),
        101_AIFLOW_v2_COMPONENT_TRANSFORMATION_PLAN.md R1 (FolderSourceAdapter)

API:
  FolderSourceAdapter(
      watch_root: Path,
      storage_root: Path,
      tenant_id: str,
      debounce_ms: int = 500,
      stable_mtime_window_ms: int = 1_000,
      max_package_bytes: int | None = None,
      glob_patterns: list[str] | None = None,  # default: ["*"]
      excluded_patterns: list[str] | None = None,  # pl. [".~*", "*.partial", "*.crdownload"]
  )

  metadata:       SourceAdapterMetadata(source_type=FOLDER_IMPORT, transport="pull",
                                        requires_ack=False)
  async fetch_next() -> IntakePackage | None    # drains internal queue
  async acknowledge(package_id)                 # idempotent (like FileSourceAdapter)
  async reject(package_id, reason)
  async health_check() -> bool                  # watch_root readable + observer alive

Kulcs invariansok:
  - Each scan produces ONE IntakePackage with ONE IntakeFile (like FileSourceAdapter).
    Multi-file folder imports belong to E3.x (N4 association) — out of scope here.
  - Observer start/stop a konstruktorban (start) es `close()` / `__aexit__` (stop).
  - Debounce: a file path-et egy bufferbe rakjuk `asyncio.sleep(debounce_ms/1000)` utan
    csak akkor dolgozzuk fel, ha azota nem jott uj esemeny.
  - Mid-write guard: `os.stat(path).st_size` + `st_mtime` ket mintat veszunk
    `stable_mtime_window_ms` kulonbseggel; ha VALTOZIK, varj tovabb.
  - Storage collision: copy-in place `storage_root/{tenant}/{package_id}/{sanitized_filename}`
    NEM hasonlit egy tukor-maphoz — eredeti fajlt hagyjuk a watch_root-ban (ez NEM ingestion-move,
    az a kovetkezo kor szovege).
  - Eliminaljuk a szivvel nem allo watchdog-lock hibakat: `try / except PermissionError/OSError`
    es napload egy strukturalt warning-gal — SOHA ne doblj ki az adapter hataran.

TILOS:
  - blocking `watchdog.observers.Observer` futtatasa event loopon → to_thread vagy dedikalt thread
  - fajl tartalmat belecopy-zni az IntakePackage-be — csak a path + sha256 + size
```
