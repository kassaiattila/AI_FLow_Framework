### Hova tartunk — Week 3 Day 11 (E3.1-A)
**N4 Associator core** — az `IntakePackage` tartalma (N files + M descriptions) assocaialasanak domain logikaja. A plan `101_*.md` N4 szerint negy mod letezik:

1. `explicit` — caller-provided mapping (`description_id → file_id` lista)
2. `filename_match` — filename regexp/glob per description
3. `order` — ordinal: description[i] ↔ file[i]
4. `single_description` — ha exactly one description, minden file-hoz kotjuk

Prefix rule: `explicit > filename_match > order > single_description`. Ha egyik mod sem tud tiszta kotest szolgaltatni → `AssociationError`.

DB side: `intake_packages.association_mode` enum column (Alembic 035), nullable-default=`single_description` a hatraflowing compat miatt.

### LEPES 2: src/aiflow/intake/associator.py

```
Cel:    N4 associator — IntakePackage → dict[file_id, description_id]
Fajl:   src/aiflow/intake/associator.py (uj)

API (javasolt):
  class AssociationMode(str, Enum):
      EXPLICIT = "explicit"
      FILENAME_MATCH = "filename_match"
      ORDER = "order"
      SINGLE_DESCRIPTION = "single_description"

  class AssociationError(AIFlowError): ...

  def associate(
      package: IntakePackage,
      *,
      mode: AssociationMode | None = None,
      explicit_map: dict[UUID, UUID] | None = None,
      filename_rules: list[tuple[str, UUID]] | None = None,
  ) -> dict[UUID, UUID]:
      ...

Precedencia: explicit > filename_match > order > single_description
Ha mode=None → auto-detect (elso sikeres mod nyer).
Ha mode=EXPLICIT de explicit_map ures → AssociationError.
Ha order mode de len(files) != len(descriptions) → AssociationError.
```

### LEPES 3: tests/unit/intake/test_associator.py

```
Cel:    ≥20 unit teszt (mode × success/failure × precedence)
Fajl:   tests/unit/intake/test_associator.py (uj)

Kotelezo scenariok:
  - [ ] EXPLICIT happy: pre-built map → egyezo dict vissza
  - [ ] EXPLICIT missing file_id → AssociationError
  - [ ] EXPLICIT missing description_id → AssociationError
  - [ ] EXPLICIT empty map → AssociationError
  - [ ] FILENAME_MATCH regex happy (invoice-*.pdf → desc_1)
  - [ ] FILENAME_MATCH no rule matches → AssociationError
  - [ ] FILENAME_MATCH ambiguous (2 rule matches ugyanarra) → AssociationError
  - [ ] ORDER mode (N files = M descriptions)
  - [ ] ORDER mode N != M → AssociationError
  - [ ] SINGLE_DESCRIPTION (1 desc → minden file-hoz)
  - [ ] SINGLE_DESCRIPTION but 2+ descriptions → AssociationError
  - [ ] No descriptions → empty map
  - [ ] No files → empty map
  - [ ] Precedencia: EXPLICIT wins over FILENAME_MATCH
  - [ ] Precedencia: FILENAME_MATCH wins over ORDER
  - [ ] Precedencia: ORDER wins over SINGLE_DESCRIPTION
  - [ ] Auto-detect (mode=None) → elso sikeres mod nyer
  - [ ] Invalid mode string → ValueError
  - [ ] EXPLICIT map points to file NOT in package → AssociationError
  - [ ] Deterministic order (dict iterated stably)
```
