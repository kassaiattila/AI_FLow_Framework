```
Cel:    POST /api/v1/sources/webhook — accepts raw body + headers, calls adapter.enqueue
Fajlok: src/aiflow/api/v1/routers/sources_webhook.py (uj)
        src/aiflow/api/v1/__init__.py (include router)
Forras: 01_PLAN/session_S55_*.md Day 9 (E2.3-B),
        101_AIFLOW_v2_COMPONENT_TRANSFORMATION_PLAN.md R1

Endpoint contract:
  POST /api/v1/sources/webhook
  Headers:
    X-Webhook-Signature:  <hex HMAC-SHA256>
    X-Webhook-Timestamp:  <unix epoch seconds>
    X-Filename:           <upload filename, required>
    Content-Type:         <mime>
    Idempotency-Key:      <optional>
  Body: raw bytes

  Returns 202 Accepted:
    { "package_id": "<uuid>", "status": "accepted" }

Error mapping (SourceAdapterError.message → HTTP status):
  "missing signature" / "invalid HMAC signature" / "replay window" → 401
  "duplicate idempotency_key"                                      → 409
  "exceeds max_package_bytes"                                      → 413
  "invalid timestamp" / "missing timestamp"                        → 400
  else                                                             → 400

Body must be read RAW (await request.body()) — do NOT parse as JSON; the
signed envelope is bytes-exact.

DI: adapter instance provided via FastAPI dependency (get_api_source_adapter)
— use a module-level singleton for Day 9; factory wiring is a later task.
```
