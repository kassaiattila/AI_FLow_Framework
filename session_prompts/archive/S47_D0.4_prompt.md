```
Cel: 4 provider ABC async interfesz.

Fajl: src/aiflow/providers/interfaces.py

ABC-k:
  1. ParserProvider:
     - metadata: ProviderMetadata (property)
     - async parse(file: IntakeFile, package_context: IntakePackage) -> ParserResult
     - async health_check() -> bool
     - async estimate_cost(file: IntakeFile) -> float

  2. ClassifierProvider:
     - metadata: ProviderMetadata (property)
     - async classify(file: IntakeFile, parser_result: ParserResult, candidate_classes: list[str]) -> ClassificationResult
     - async health_check() -> bool

  3. ExtractorProvider:
     - metadata: ProviderMetadata (property)
     - async extract(file: IntakeFile, parser_result: ParserResult, config: dict) -> ExtractionResult
     - async health_check() -> bool

  4. EmbedderProvider:
     - metadata: ProviderMetadata (property)
     - dimensions: int (property)
     - async embed(texts: list[str]) -> list[list[float]]
     - async health_check() -> bool

FONTOS:
  - abc.ABC + @abstractmethod
  - TYPE_CHECKING guard circular import-okhoz
  - Result types (ParserResult, ClassificationResult, ExtractionResult) string forward ref
    (meg nincsenek implementalva Phase 1a-ban)
  - Minden metodus async
```
