## FELADATOK

### LEPES 1: Uj helper modul

```
Cel:    Egyetlen helyen definialni a canonical source event emittert.
Fajl:   src/aiflow/sources/observability.py  (uj)
Forras: 01_PLAN/session_S73_v1_4_2_phase_1c_kickoff.md Day 3, plan sor 98-113

Tartalom (skeleton):
  from typing import Any, Literal
  import structlog
  from aiflow.intake.package import IntakePackage

  logger = structlog.get_logger(__name__)

  CanonicalEvent = Literal["source.package_received", "source.package_rejected"]

  def emit_package_event(
      event: CanonicalEvent,
      package: IntakePackage,
      source_type: str,
      **extra: Any,
  ) -> None:
      """Emit the canonical source-adapter package event.

      Event shape (stable contract for dashboards & alerts):
        - event:        source.package_received | source.package_rejected
        - package_id:   str (UUID)
        - tenant_id:    str
        - source_type:  file|folder|batch|api|email
        - **extra:      non-PII context (reason, count, ...). NO password / hmac / signature / body.
      """
      logger.info(
          event,
          package_id=str(package.package_id),
          tenant_id=package.tenant_id,
          source_type=source_type,
          **extra,
      )

Module docstring top: deprecation-timeline note for old per-adapter events
(keep as secondary emission for 1 minor release, remove in v1.5.x).
```

  src/aiflow/sources/email_adapter.py   ~377 (acknowledged) + ~391 (rejected)

Pattern per site (example for file_adapter ack):
  # BEFORE:
  logger.info("file_adapter_acknowledged", package_id=str(package_id))

  # AFTER:
  logger.info("file_adapter_acknowledged", package_id=str(package_id))  # legacy, deprecated v1.5.x
  emit_package_event(
      "source.package_received",
      package,
      source_type="file",
      # no PII extra
  )

Source_type mapping:
  file_adapter   → "file"
  folder_adapter → "folder"
  batch_adapter  → "batch"
  api_adapter    → "api"
  email_adapter  → "email"

FONTOS PII tilalom:
  - NE adj hozza "password", "hmac_secret", nyers "signature", email "body", raw content
  - Ha kell context → aggregalt (count, reason_code), ne payload
