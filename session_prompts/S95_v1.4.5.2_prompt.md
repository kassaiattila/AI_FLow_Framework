### LÉPÉS 2 — `MultiSignalRouter` (new module)

Fájl: `src/aiflow/routing/router.py` (új modul: `src/aiflow/routing/__init__.py` + `router.py`)

```python
class MultiSignalRouter:
    def __init__(self, policy_engine: PolicyEngine, registry: ProviderRegistry) -> None: ...

    async def decide(
        self,
        package: IntakePackage,
        file: IntakeFile,
    ) -> RoutingDecision: ...
```

**Rule set (S95 scope, szándékosan szűk):**

1. `size_bytes <= 5_000_000` AND `mime_type ∈ {application/pdf (born-digital hint), application/vnd.openxmlformats-officedocument.wordprocessingml.document, text/plain, text/html, text/markdown}` → `chosen_parser="unstructured_fast"`, `fallback_chain=["docling_standard"]`.
2. Minden más (nagy PDF, scan-gyanús, DOCX/XLSX/PPTX nagy fájl, képek) → `chosen_parser="docling_standard"`, `fallback_chain=[]`.
3. Ha `policy.cloud_ai_allowed == False` ÉS a MIME cloud-only (nincs self-hosted path) → `chosen_parser="skipped_policy"`, `reason="cloud_ai_disallowed_for_mime"`.

**Born-digital vs scan detekció** nincs S95-ben. Csak MIME + size gate. Scan-aware routing S96 scope.

### LÉPÉS 4 — `DocumentExtractorService.extract_from_package()` router wiring

Módosítsd `src/aiflow/services/document_extractor/service.py`:

- Az `extract_from_package()` szignatúra kapjon egy új paramétert: `router: MultiSignalRouter | None = None`.
- Ha `router` adott: file-onként `decision = await router.decide(pkg, file)` → `parser = registry.get_parser(decision.chosen_parser)(...)` → parse → ExtractionResult (plusz `structured_fields["routing_decision_id"] = decision.package_id+file_id`-ből származott uuid, vagy épp a `decision.reason`).
- Ha nincs router: marad az S94 viselkedés (default DoclingStandardParser). **Ne tördelj backward-compat tesztet.**
- A `_extract_single_file` kiegészül: ha `decision.chosen_parser == "skipped_policy"` → ugyanaz az ExtractionResult mint S94-ben, de `structured_fields["skip_reason"] = decision.reason`.

### LÉPÉS 5 — Alembic 038: `routing_decisions` tábla

Fájl: `alembic/versions/038_<slug>_routing_decisions.py`

```sql
CREATE TABLE routing_decisions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    package_id UUID NOT NULL,
    file_id UUID NOT NULL,
    tenant_id TEXT NOT NULL,
    chosen_parser TEXT NOT NULL,
    reason TEXT NOT NULL,
    signals JSONB NOT NULL DEFAULT '{}'::jsonb,
    fallback_chain JSONB NOT NULL DEFAULT '[]'::jsonb,
    cost_estimate NUMERIC(10,6) NOT NULL DEFAULT 0,
    decided_at TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
);
CREATE INDEX ix_routing_decisions_package ON routing_decisions (package_id);
CREATE INDEX ix_routing_decisions_tenant ON routing_decisions (tenant_id);
```

Zero-downtime rule: minden oszlop nullable vagy default, semmi NOT NULL backfill nélkül. `downgrade()` csak DROP TABLE.
